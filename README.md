# Smart Energy Meter

## Overview

Developed a Smart Energy Meter using ESP32 for real-time monitoring of voltage, current, power, and energy consumption through IoT.

## Features

* Real-time voltage and current monitoring
* Power and energy consumption calculation
* IoT-based remote monitoring
* Energy usage tracking
* Wireless data transmission using Wi-Fi

## Components Used

* ESP32
* Voltage Sensor
* Current Sensor
* LCD Display
* Relay Module
* Blynk IoT Platform

## Software Used

* Arduino IDE
* Blynk IoT
* Embedded C

## Applications

* Smart homes
* Energy management systems
* Industrial energy monitoring
